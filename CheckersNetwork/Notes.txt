Порядок
Игровое поле состоит из игровых клеток
Шашки игрока располагаются поверх игровых клеток
Шашки ходят только в сторону края противника ( для этого хранится их нарпавление хода )

Ход совершается в два действия:
первый клик на шашку которая будет ходить
второй клик на место куда она будет ходить
прерывание хода совершается кликом в любое другое не подходящее для хода место.

Логика хода:
Клик на шашку - проверка на состояние дамки - проверка соседних клеток на поле (цикличная проверка последующих) - создание подсвеченных участков возможных для хода - клик на подсвеченный участок
